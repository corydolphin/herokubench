hbench

Herokubench allows you to easily load test websites, from the cloud, for free. Use hundreds of free dynos on Heroku to run apache-bench, in the same way you would run it locally.

Installing Heroku Bench

gem install herokubench

Getting Started

Using HerokuBench is as easy as using apache bench.

hbench ab -c 100 -n 10000 http://www.google.com/

Making it Rain

Hbench shines when running a load test with hundreds of thousands of requests per second. Unlike running apache-bench from your own PC, Hbench distributes the requests across a number of dynos, allowing far higher throughput. Testing with 10,000 concurrent connections is the same as testing with 100.

hbench ab -c 10000 -n 1000 http://www.google.com/

Example usage

Using Hbench on a simple nodejs server

hbench ab -c 1000 -n 10000 http://nodejssimple.herokuapp.com/
